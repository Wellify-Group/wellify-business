# Email Templates для WELLIFY Business

## Описание

Этот каталог содержит HTML-шаблоны для писем подтверждения email, которые полностью отключают автоматическую инверсию темной темы в почтовых клиентах.

## Файлы

- **`email-confirmation.html`** - Шаблон для Resend (использует переменную `{{link}}`)
- **`email-confirmation-supabase.html`** - Шаблон для Supabase (использует переменную `{{ .ConfirmationURL }}`)

## Особенности

✅ **Полное отключение инверсии темной темы**
- Мета-теги `color-scheme: only light`
- CSS правила для принудительного белого фона
- Медиа-запросы для переопределения темной темы

✅ **Inline стили**
- Все стили встроены в HTML для максимальной совместимости
- Поддержка всех основных почтовых клиентов

✅ **Чистый дизайн**
- Белый фон (#ffffff)
- Нет градиентов, прозрачностей, backdrop-filter
- Простые, надежные цвета

## Применение

### Для Resend:

1. Откройте **Resend Dashboard** → **Emails** → **Templates**
2. Создайте новый шаблон или отредактируйте существующий
3. Скопируйте содержимое файла `email-confirmation.html`
4. Вставьте в редактор шаблона
5. Убедитесь, что переменная `{{link}}` соответствует вашей настройке Resend

### Для Supabase:

1. Откройте **Supabase Dashboard** → **Authentication** → **Email Templates**
2. Выберите шаблон **"Confirm signup"**
3. Скопируйте содержимое файла `email-confirmation-supabase.html`
4. Вставьте в редактор шаблона
5. Сохраните изменения

## Тестирование

После применения шаблона протестируйте отправку письма:

1. **Resend Dashboard** → Test Email
2. **Supabase Dashboard** → Test Email

Проверьте отображение в следующих почтовых клиентах:

- ✅ iPhone Mail (dark / light mode)
- ✅ Gmail (mobile / desktop)
- ✅ Apple Mail
- ✅ Outlook
- ✅ Android Mail

**Ожидаемый результат:** Фон всегда остается белым (#ffffff), независимо от системной темы устройства.

## Структура шаблона

```
- <head> - Мета-теги и стили для отключения инверсии
- <body> - Основной контейнер с белым фоном
  - Wrapper table - Внешняя таблица
    - Container table - Внутренняя таблица (max-width: 600px)
      - Logo/Header - Логотип и заголовок
      - Content - Основной контент
        - Заголовок и текст
        - Кнопка подтверждения
        - Ссылка-запасной вариант
        - Footer - Подвал
```

## Цвета

- **Фон:** `#ffffff` (белый)
- **Текст:** `#000000` (черный)
- **Вторичный текст:** `#666666` (серый)
- **Кнопка:** `#2D6BFF` (синий)
- **Ссылка:** `#2D6BFF` (синий)
- **Граница:** `#e5e7eb` (светло-серый)

## Поддержка

Если письмо все еще инвертируется в каком-то почтовом клиенте:

1. Убедитесь, что все мета-теги присутствуют в `<head>`
2. Проверьте, что все стили inline (не внешние CSS файлы)
3. Убедитесь, что нет градиентов или прозрачностей
4. Проверьте, что `!important` используется для критических стилей
